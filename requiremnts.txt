Installation

Make sure you have installed Python 2.6.5 or higher, pip, virtualenv,postgres
Run the following below command to install dependencies for PostgreSQL to work with Django
sudo apt-get install libpq-dev python-dev
Run the following commands 
virtualenv venv
source venv/bin/activate
pip install django
pip install psycopg2
pip install pillow

Configuration
sudo su - postgres
createdb systersdb1
createuser -P
psql
GRANT ALL PRIVILEGES ON DATABASE  systersdb TO (user that was created in previous command);
